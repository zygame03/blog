# ZYGame的个人博客

简单的个人博客，算是个练手项目，顺便记录生活。目前只计划实现基础的主页及文章展示功能，后续逐步完善网站，提供更丰富的互动内容。

## 主要内容

- 一些学习笔记，记录成长与进步

- 生活随笔，偶尔写写日常生活和感悟
